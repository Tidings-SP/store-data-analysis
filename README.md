# 🧸 Maven Toys Sales Analysis & Sales Prediction

## 📌 Project Overview

This project is an end-to-end **Data Analytics** project based on the **Maven Toys Sales Dataset**. The objective is to analyze the sales performance of a toy retail chain, identify business insights, build interactive dashboards, and develop a simple machine learning model to predict future sales.

The project follows the complete data analytics lifecycle:

1. Business Understanding
2. Data Understanding
3. Data Cleaning
4. Data Integration
5. Exploratory Data Analysis (EDA)
6. Business Insights
7. Feature Engineering
8. Sales Prediction
9. Dashboard Creation
10. Final Business Recommendations

---

# 🎯 Objectives

* Analyze sales performance across stores and products.
* Identify top-performing and underperforming products.
* Evaluate inventory trends.
* Understand seasonal sales patterns.
* Generate actionable business insights.
* Build a simple prediction model for future sales.
